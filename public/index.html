PROJECTS

tshirt-creator

T-shirt creator is a web app for creating images of t-shirts for sellers of shirts. The app allows you to upload your own artwork, position and scale the artwork on the shirt and change shirt colors. You can then download the images for use on your own site.

The main goal of the app is to make it quicker and easier for people to create images of t-shirts rather than having to manually create them. The app is still a work in progress but is created with React and will soon have Firebase integration for storing images and managing users.

amateur-radio.school

I created a website to help people practice for the amateur radio exams. The website allows you to practice for all three classes of exams: Technician, General and Amateur Extra.

The project was created with React. The question data is hosted on Contentful and the client side is hosted on Netlify. It allows a user to take an exam and then see their results at the end.
