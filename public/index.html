I have created quite a few different projects. I like to try using different technologies. Some projects are created with just vanilla JavaScript and others are created using Vue or React. Some of them are still works in progress.

  • amateur-radio.school

    I created a website to help people practice for the amateur radio exams. The website allows you to practice for all three classes of exams: Technician, General and Amateur Extra.

    Converting The Data

    The FCC provides a pool of questions that are to be used on the exams. I wrote some python scripts to convert the question pool data, which is provided in text and PDF formats, into JSON. I then used the headless CMS Contentful to host the question data. The client side of the website is hosted on Netlify.

    The project was created with React and is still a work in progress but is currently functional. It allows a user to take an exam and then see their results at the end.

  • Resistor Calculator

    This is a resistor calculator or decoder. The resistance in a resistor can be calculated by looking at the colored bands. This little app allows you to enter the colors of the bands and tells you the resistance in Ohms.

    It was created with just Vanilla Javascript, html and css. The Javascript files can be packaged using either Webpack or parcel.

  • Fallout Password Hacking Game

    This is my attempt at a Fallout hacking minigame. If you have ever played any of the fallout games then you have most likely seen the hacking minigame where you have to guess a password. This is still a work in progress.

    This project is created with Vanilla Javascript on the frontend. There is an express server that runs the word server. The word server just returns a list of words of a certain length to be used in the game.

    As I said above this is still a work in progress. I plan on removing the word server and just having a list of words on the client side.
