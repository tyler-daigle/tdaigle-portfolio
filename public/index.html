  • Budget Predictor

    Budget Predictor is an app created with Vue which allows you to enter transactions (mortgage payments, salary, rental income, etc.) and calculates how much money you will have at the end of each month. I don't have much experience with Vue so I decided to create this project to learn more about Vue.

    Some features of Budget Predictor:

    • Ability to edit, delete, or add transactions.
    • App is responsive on different screen sizes.
    • Custom dialog box for adding/deleting/editing transactions.
